# IR Sensor PCB Design

## Overview
This project is an IR Sensor PCB designed using KiCad. The project includes schematic capture, PCB layout, and routing.

## Features
- IR transmitter and receiver interface
- Custom PCB layout
- Designed using KiCad
- Compact design

## Software Used
- KiCad

## Files Included
- IR SENSOR.kicad_sch (Schematic)
- IR SENSOR.kicad_pcb (PCB Layout)
- IR SENSOR.kicad_pro (Project File)
